# PyTorch MNIST Classifier 🔥

## 📌 Overview
This project implements a neural network for handwritten digit classification using PyTorch.

It demonstrates:
- Dataset handling using DataLoader
- Model building using nn.Module
- Training using autograd
- Evaluation on test data

## ⚙️ Tech Stack

- PyTorch
- Torchvision
- Matplotlib

---

## 📊 Model Architecture

Fully Connected Neural Network:

- Input: 28×28 (flattened to 784)
- Hidden Layer 1: 128 neurons (ReLU)
- Hidden Layer 2: 64 neurons (ReLU)
- Output: 10 classes
